drwxr-xr-x 🎵 recomendify / Python · Grafos · PageRank · BFS · DFS · Backtracking
❯

Motor de recomendación de música construido sobre un grafo bipartito ponderado usuario–canción–playlist con datos reales de Spotify. Implementación propia de algoritmos clásicos de teoría de grafos aplicados a un problema real de recomendación, sin librerías externas de grafos.

  • Grafo dirigido/no dirigido implementado desde cero con diccionarios de adyacencia
  • PageRank clásico convergente — 52 iteraciones, d=0.85 según paper de la cátedra
  • PageRank personalizado via random walk — 10.000 iteraciones, walk length 20 — para recomendaciones por contexto
  • BFS para camino más corto entre canciones con reconstrucción de path y playlist
  • DFS con backtracking para detección de ciclos de longitud exacta N
  • Heap para ranking eficiente O(k log n) — top-K canciones o usuarios
